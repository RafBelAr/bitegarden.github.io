Universal Plugin Manager for SonarQube™

Provides a user-friendly interface to manage your plugins

For Administrators by Developers

How many times you want to install a custom plugin into your existing SonarQube™ installation and you have to wait till the operations team copy the "file" to the server into your "extensions/plugins" folder?

This will be the last time to do that!

The Universal Plugin Manager (UPM) is a tool for administering plugins in SonarQube™.

You can use the UPM to install single plugins (jar files) through SonarQube™ administration interface.

You won't need to access SonarQube™ server filesystem anymore to install a custom plugin!

One plugin to rule them all!
