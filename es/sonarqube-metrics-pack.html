Metrics Pack Plugin for SonarQube™

¡Nuevas métricas para SonarQube™!

Obtén un paquete de métricas adicionales para SonarQube™, ¡gratis!

Este plugin de SonarQube proporciona información adicional a tus análisis de SonarQube™.

Metrics Pack hace regresar a SonarQube™ métricas como el RCI (Rules Compliance Index), y valores sobre el código muerto o el ratio de deuda técnica total

Descripción de las métricas

Rules Compliance Index (RCI)

Métrica basada en el valor ponderado de las evidencias y la cantidad de líneas de código

Dead Code Issues

Esta métrica incluye el número de problemas relacionados con el código muerto. Las reglas que se aplican a la métrica se pueden personalizar a través de la configuración del complemento.

Estimated Development Cost

Según la configuración del coste para desarrollar una línea de código (LOC), calculamos el coste total de desarrollo estimado para sus recursos de SonarQube

Total Technical Debt Ratio

SonarQube™ calcula el índice de deuda técnica solo para evidencias del tipo code smell. Con esta nueva métrica conocerá su índice de deuda técnica general, incluidos los costes de reparación de bugs y vulnerabilidades.

Lines of Code per File

Solo por diversión, ¿alguna vez te preguntaste las líneas de código promedio de tu proyecto? ¡Aqui lo tienes!
