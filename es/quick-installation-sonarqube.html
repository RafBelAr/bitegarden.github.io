Guía de instalación rápida

Esta guía te ayuda a instalar los plugins de SonarQube™

Has descargado un archivo "jar" que contiene nuestro plugin.

Utiliza bitegarden UPM para instalar el plugin directamente desde la interfaz del usuario de administración o copiar el archivo jar en la carpeta de plugins de SonarQube™: 

$SONAR_HOME/extensions/plugins/

Reinicia el servidor de SonarQube™ y disfruta de las nuevas funcionalidades

Dependiendo del plugin es posible que necesite ejecutar un nuevo análisis para calcular los datos.

Encontrará las principales características de los plugins en la sección "Más..." en los menús de espacios globales o de proyectos.


NOTE: El plugin incluye la licencia de prueba durante 2 semanas. No tienes que configurar nada más.
