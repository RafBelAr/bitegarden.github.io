SonarQube™ plugins

Construimos plugins de SonarQube™ para mejorar tu ecosistema de inspección continua

  • Security Plugin for SonarQube™

    Security Plugin for SonarQube™

    Proporciona información sobre estándares de seguridad como OWASP en un apartado dedicado en cada proyecto de SonarQube™, incluyendo factores de riesgo y vulnerabilidades de seguridad

  • Overview Report Plugin for SonarQube™

    Overview Report Plugin for SonarQube™

    Permite agregar proyectos en SonarQube™ para tener una vista general de las métricas de calidad de código en una única vista

  • SQALE Plugin for SonarQube™

    SQALE Plugin for SonarQube™

    El método SQALE aplicado en SonarQube™ permite gestionar la deuda técnica gracias a un panel dedicado para gestionar las características del modelo

  • Report Plugin for SonarQube™

    Report Plugin for SonarQube™

    Genera un informe a nivel proyecto con la información sobre el mismo y personalízalo. Reporte generado en ODT y disponible para descargar o enviar por email

  • Metrics Pack Plugin for SonarQube™

    Metrics Pack Plugin for SonarQube™

    Pack de métricas adicionales para SonarQube™. Incluye métricas relacionadas con RCI, dead code, etc.

  • Rating Report Plugin for SonarQube™

    Rating Report Plugin for SonarQube™

    Añade un informe adicional usando la información de las etiquetas en SonarQube™. Podrás valorar el estado de la calidad desde diferentes parámetros.

  • Project Tag Plugin for SonarQube™

    Project Tag Plugin for SonarQube™

    Establece tags/etiquetas automáticas en el onboarding de proyectos en SonarQube™
