¡Gracias por descargar nuestros productos!

Tu descarga se abrirá en una nueva ventana del navegador.

Si ocurre algún problema puedes intentarlo a través de este enlace directo.

Los primeros pasos

Acabas de descargar un fichero "jar" auto ejecutable que contiene el producto para SonarCloud™. Requiere Java 8 o posterior.

Ejecuta el jar con la opción --help para ver todas las opciones de uso (cambia bitegarden-product.jar por el fichero que acabas de descargar): 

java -jar bitegarden-product.jar --help


NOTA: El producto incluye una licencia de prueba de 2 semanas para su evaluación. No tienes que configurar ninguna licencia si solo quieres evaluarlo.

¿Necesitas ayuda?

¿Tienes alguna duda mientras evalúas nuestro producto? ¿Has encontrado un bug en nuestros plugins o quieres sugerirnos una mejora? Abre un ticket en nuestro Jira Service Desk y contactaremos contigo a la mayor brevedad posible.
