Grazie per aver scaricato il plugin

Il download si aprirà in una nuova finestra del browser.

Se si verifica un problema, puoi provarlo tramite questo collegamento diretto.

Installazione

Hai appena scaricato un file "jar" che contiene il plugin.

Utilizza bitegarden Universal Plugin Manager per installarlo direttamente dall'interfaccia di amministrazione o copia il file jar nella cartella dei plugin SonarQube™: 

$SONAR_HOME/extensions/plugins/

Riavvia SonarQube™ e goditi la nuova funzionalità.

A seconda del plugin potrebbe essere necessario eseguire una nuova analisi per elaborare tutti i dati.

Troverai le principali funzionalità dei plugin nella sezione "More..." nella barra dei menu globale o del progetto.


NOTA: Il plugin include una licenza di prova di 2 settimane. Non devi configurare nient'altro.

Richiedi supporto

Hai domande sul nostro prodotto? Hai trovato un bug nel plugis o vuoi suggerirci come migliorare? Apri un ticket nel nostro Jira Service Desk e ti ricontatteremo nel più breve tempo possibile.
