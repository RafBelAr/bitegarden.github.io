Grazie per il tuo acquisto

Abbiamo appena ricevuto il tuo pagamento per il plugin bitegarden

Come ricevere la tua licenza

Per generare la chiave di licenza abbiamo bisogno che tu ci fornisca i seguenti dati:

  • - dati aziendalia per generare la fattura (ragione sociale, indirizzo, partita IVA, ...)

  • - ID server della tua istanza diSonarQube

Non appena ci riceveremo queste informazioni, ti invieremo la chiave della licenza e la fattura.


INVIA RICHIESTA

Dove trovo il mio ID server?

Il tuo ID server viene utilizzato per identificare in modo univoco la tua istanza SonarQube. Si trova nella pagina Administration > System nella sezione delle Informazioni di sistema.
