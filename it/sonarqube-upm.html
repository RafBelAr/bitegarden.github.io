Universal Plugin Manager per SonarQube™

Gestisci i tuoi plugin attraverso l'interfaccia di amministrazione

Per amministratori da sviluppatori

Quante volte hai desiderato installare un plugin personalizzato sulla tua istanza SonarQube™ e hai dovuto aspettare che il Team di Operation/Sistemi copiasse il "file" sul tuo server all'interno della cartella "estensioni/plugin"?

Questa sarà l'ultima volta che lo farai! L'Universal Plugin Manager (UPM) è uno strumento per la gestione dei plugin in SonarQube ™ tramite interfaccia.

È possibile utilizzare l'UPM per installare singoli plugin (file "jar") tramite l'interfaccia di amministrazione di SonarQube ™.

Con pochi clic puoi navigare tra i plugins di bitegarden e i tuoi plugins personalizzati senza dover uscire dall'interfaccia di amministrazione di SonarQube™.

Il caricamento del file installa il plugin come file JAR. Il file deve contenere tutte le risorse e il codice necessari per l'esecuzione. Il caricamento dei file consente di installare plugins personalizzati o qualsiasi altro plugin non disponibile nel Marketplace SonarQube ™. Puoi anche usarlo per installare plugin per istanze SonarQube™ che operano dietro un firewall e disconnesse da Internet.

Non avrai più bisogno di accedere al filesystem del server SonarQube™ per installare nuovi plugin!

SCARICA GRATUITAMENTE

Plugins di bitegarden

Universal Plugin Manager per SonarQube™ include una nuova sezione per gestire tutti i plugin sviluppati da bitegarden.

In questo modo, semplicemente installando l'Universal Plugin Manager potrai per gestire e installare tutti i nostri plugin. Potrai anche aggiornare le versioni o eliminare quelle che non usi dall'interfaccia di amministrazione.

Inoltre, include anche la funzionalità di autoaggiornamento. Ti notificherà la disponibilità di ogni nuova versione. Potrai poi aggiornare i plugin direttamente dalla pagina di amministrazione dell'Universal Plugin Manager.

Plugin gestiti dall'utente

Puoi installare qualsiasi plugin caricandolo direttamente sulla tua piattaforma SonarQube™. Devi utilizzare questo metodo per installare plugin che non sono disponibili nella sezione in primo piano o nel Marketplace ufficiale, come i plugins sviluppati da te o se desideri installare la vecchia versione di un plugin.

Avviso di sicurezza: i plug-in sono molto potenti e possono modificare il comportamento di quasi tutte le funzionalità della piattaforma SonarQube ™. È molto importante fidarsi dei plugin prima di installarli. Dovresti sempre fare attenzione da dove proviene il plugin che desideri installare (e soprattutto da chi proviene).
