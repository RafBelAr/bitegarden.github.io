Grazie per aver scaricato il plugin

Il download si aprirà in una nuova finestra del browser.

Se si verifica un problema, puoi provarlo tramite questo collegamento diretto.

Installazione

Hai appena scaricato un file "jar" che contiene il plugin (standalone app) per SonarCloud™. Richiede Java 8 o una versione successiva.

Esegui il comando con l'opzione --help per vedere come usarlo (cambia bitgarden-product.jar nel file che hai appena scaricato): 

java -jar bitegarden-product.jar --help


NOTA: il prodotto include la la licenza di prova per 14 giorni. Non hai bisogno di settare nient'altro per utilizzare il plugin.
