Report Plugin per SonarQube™

Scarica il report del tuo progetto da SonarQube in PDF o crea un report personalizzabile in ODT

PROVA GRAUITA    ACQUISTA ORA

Genera un report con metriche di SonarQube e scaricalo con un click

Hai mai avuto bisogno di esportare un report da SonarQube™? Vuoi esportare l'informazione di cui hai bisogno in maniera semplice e rapida? Scarica il tuo report di SonarQube in PDF o in un file ODT personalizzabile grazie al Report Plugin per SonarQube™.

Con questo plugin potrai scaricare i report in PDF con le informazioni più importanti sulla qualità del codice del tuo progetto. Come? Direttamente dall'interfaccia utente con un click, senza nessuna configurazione aggiuntiva. E' semplice e rapido.

In più potrai creare dei reports completamente personalizzabili attraverso un file ODT (Open Document). I report racchiudono le metriche del tuo progetto di SonarQube™ e lo presentano in un documento editabile che successivamente potrai modificare per aggiungere i tuoi commenti o ciò di cui hai bisogno per il tuo rapporto.

PROVA GRATUITA ACQUISTA ORA

Il Report Plugin per SonarQube™ ti consente di risparmiare tutto il tempo che ti costerebbe generare un report sullo stato della qualità dei tuoi progetti. Ora sarai in grado di scaricare un report di SonarQube in PDF, o ODT personalizzabile, in pochi secondi, ogni volta in cui ne hai bisogno e con le informazioni del tuo progetto in tempo reale.

Caratteristiche

CARATTERISTICHE PRINCIPALI DEL PLUGIN

- Report esecutivo riassuntivo in PDF con tutte le metriche più importanti in una pagina.
- Report riassuntivo delle evidenze in PDF che include bugs, vulnerabilities e code smells.
- Report completo che include TUTTI i bugs, le vulnerabilità e i code smells del tuo progetto.
- Personalizxa completamente il modello ODT (intestazioni, tabelle, testi, titoli, copertina, immagini, ...) per creare un report completamente personalizzato

INOLTRE...

- Permette di generare reports di qualsiasi ramo del tuo progetto.
- Permette di configurare il logo al piè di pagina per personalizzare i tuoi report.
- E' compatibile con tutti i linguaggi e la tecnologia di SonarQube™, includendo tutti i plugins di terze parti.
- I report si generano in tempo reale e si possono scaricare dall'interfaccia utente di SonarQube™ o attraverso un endpoint di un servizio web per integrarlo con altri strumenti.
