Guida di installazione rapida

Questa guida ti aiuta a installare il nostro SonarQube™ Plugin

Hai scaricato un file "jar" contenente il nostro plugin.

Usare bitegarden UPM per installare il plugin direttamente dall'interfaccia utente di amministrazione o copiare il file jar nella cartella SonarQube™plugins: 

$SONAR_HOME/extensions/plugins/

Riavvia il server SonarQube™ e goditi le

A seconda del plugin potrebbe essere necessario eseguire una nuova analisi per calcolare i dati.

Troverai le caratteristiche principali dei plugin nella sezione "Altro..." nei menu degli spazi globali o del progetto.
