Guida all'installazione rapida

Questa guida ti aiuta a iniziare a utilizzare i nostri prodotti SonarCloud

Hai scaricato un file eseguibile automatico "jar" contenente il prodotto (app standalone) per SonarCloud™. Richiede Java 8 o successivo per essere eseguito.

Run the command with the --help option to see how to use it (change bitegarden-product.jar to the file you have just download): 

java -jar bitegarden-product.jar --help


NOTA: Il prodotto includela licenza di prova per 2 settimane per la valutazione. Non è necessario impostare alcuna chiave di licenza.

Serve aiuto?

Hai domande durante il periodo di prova? Hai trovato un bug nei nostri plugin o vuoi per suggerire un miglioramento? Aprire un biglietto nel nostro Jira Service Desk e ti risponderemo il prima possibile.
