Metrics Pack Plugin per SonarQube™

Un pack di nuove metriche per SonarQube™!

Tieni sotto controllo la qualità del tuo codice con il pack di metriche gratuite

Questo plugin SonarQube fornisce informazioni aggiuntive alle tue analisi SonarQube™.

Metrics Pack fa ritornare in SonarQube™ metriche come l'RCI (Rules Compliance Index) e i valori sul dead code o il rapporto di indebitamento tecnico totale.

SCARICA GRATUITAMENTEE

Descrizione delle metriche

Rules Compliance Index (RCI)

Metrica basata sul valore ponderato dell'evidenza e sul numero di righe di codice.

Dead Code Issues

Questa metrica include il numero di problemi relativi al codice morto. Le regole che si applicano possono essere personalizzate tramite le impostazioni del plug-in.

Estimated Development Cost

In base alle impostazioni dei costi per sviluppare una riga di codice (LOC), viene calcolato il costo di sviluppo totale stimato per le tue risorse di SonarQube.

Total Technical Debt Ratio

SonarQube™ calcola il rapporto di indebitamento tecnico solo per i problemi classificati come code smells. Con questa nuova metrica conoscerai il tuo debt ratio complessivo, compresi i costi di riparazione di bug e vulnerabilità.

Lines of Code per File

Tanto per divertirti, ti sei mai chiesto qual è la media delle righe di codice del tuo progetto? Scoprilo con il Metrics Pack per SonarQube!

Espandi le capacità di SonarQube™ con le nuove metriche!
