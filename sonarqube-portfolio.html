SonarQube Portfolio Plugin

Provides a big picture of your SonarQube instance with all the aggregated measures in a single view

Code Quality Overview for your portfolio

We love SonarQube... We have a lot of projects... but... what about your whole instance? what about a team? what about a specific portfolio of projects? Did you ever though about getting a big picture?

With this plugin you will be able to manage your views or portfolios in SonarQube instance. Create new portfolios/views inside SonarQube user interface and get an overview of all the aggregated quality metrics. By default if no portfolio is defined, it will display an overview of all your projects in your SonarQube instance

On a single page you will get all aggregated metrics, including your leak period.

You will be able to see total number of bugs, vulnerabilities and code smells, along with size metrics, duplications or coverage.

Features

This plugin provides a global page with all the collected metrics and global ratings for the portfolios or your whole instance. This includes the project distribution for each rating (size, reliability, security and maintainability) and the aggregated measures.

Global ratings are computed as follows:

- Size rating: this is the sum of all lines of code of all the projects. The rating is computed exactly the same as a individual project. See Size Rating for more information.

- Reliability, Security and Maintainability rating: these ratings are computed by giving values from 1 to 5 to each project depending on the rating (A=1 ... E=5). Then we sum all the values and compute the average number. This number is rounded to the nearest value from 1 to 5. This value will be the global rating for your projects.

Define your portfolios in SonarQube

Go to plugin settings (Administration > General Settings > Portfolio). The portfolio section allows you to configure your custom portfolios in SonarQube.

You just need to enter your portfolio name and a comma separated list of resource keys to be included in the portfolio.

You will be able to see total number of bugs, vulnerabilities and code smells, along with size metrics, duplications or coverage, for each defined portfolio in the global page.

If no portfolio is defined, then the plugin will compute all your projects and will provide an overview summary.
